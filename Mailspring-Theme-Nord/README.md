# Mailspring Nord Theme

This is my attempt at a Nord theme for Mailspring. Based on the [Nord](https://www.nordtheme.com/) color scheme.

![screenshot](/screenshot/nord-mailspring.png)

## Install

1. Download or clone this repository.

2. In mailspring: `Edit -> Install Theme -> /path/to/Mailspring-Theme-Nord`.
